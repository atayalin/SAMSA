About Project

How to Use?

Emotions are things that specify a person’s acts, feelings, Emotions play a significant role in our lives, influencing our thoughts, actions, and relationships. Understanding and examining emotions can provide valuable insights and foresight in various areas. Exploring emotions through research and analysis empowers us to make informed choices, cultivate empathy, and create a more harmonious and fulfilling existence. Since emotions are this much significant in society, in our project we are examining movie/tv series scripts with NLP to find out the emotions. We are doing our research specifically on movie/tv series scripts because one of the best reflections of real life and real human emotions that we can observe is from movies/tv series.

Natural Language Processing (NLP) and speech recognition are widely utilized artificial intelligence (AI) applications that have become increasingly prevalent in various domains. The objective of this project is to explore the field of Natural Language Processing (NLP) and Speech Recognition and implement them to work on emotions. Our aim was to create a Natural Language Processing model, and use that model on the script that we obtained by already in use Speech Recognition tools. Eventually, our aim was to obtain a model which we can use to predict the emotions in a movie/tv series script. As a result we used the obtained modules on scripts to visualize the emotions. For the visualization we are using python tools for pie charts, bar charts, etc. As our final product we designed an NLP model, used already generated Speech Recognition models with some changes and gathered them in our website.

Allowed extensions are .json, .wav, .mp3 and .mp4. If the input file's extension is incorrect, it will return the related error message.

  1. The input box at the bottom of the page is used to select the input files. Click it and select the file that is going to be uploaded or Drag&Drop to the same place.
  2. Press the Upload button that is on the right side of the upload box. If you are not sure of the file, click the Cancel button and reselect the file. The file will respond with a "finished" message if it is successfully uploaded. Otherwise, it will return the related error.
  3. If the upload operation is finished correctly, you can see the uploaded file, processed information, resulting charts, and any other information on the database and movie pages.

If the file is still in the SpeakerDiarization process you cannot be able to view the resulting JSON files, charts, speakers, etc.
